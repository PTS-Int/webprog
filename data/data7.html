8. Sparta

Project Name: Sparta
Subject: 2014S1 - Distributed System Essentials
Details: The final project of Distributed System Essentials subject. It's a chat application like "Line" which user can talk with each other in ordered message. Also this application can create group which everyone can talk together while the message received by everyone must be in the same order.
Role: Designer, Tester
